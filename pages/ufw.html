UFW & Güvenlik

Firewall kuralları ve port yönetimi

⚠
UFW'yi etkinleştirmeden önce port 22 (SSH) mutlaka açık olmalı. Aksi halde sunucuya erişim kalıcı olarak kesilebilir.
UFW Komutları
mevcut kuralları gör
ufw status
Aktif firewall kurallarını ve UFW durumunu göster.
port aç
ufw allow PORT
Belirtilen porta gelen trafiğe izin ver. Örnek: 22, 80, 443, 5000.
port kapat
ufw deny PORT
Belirtilen portu kapat. Uygulama production'a geçince test portunu kapat.
temel kurulum (ilk kez)
ufw allow 22 ufw allow 80 ufw allow 443 ufw enable
sıra önemli
SSH, HTTP ve HTTPS portlarını aç, sonra UFW'yi aktif et.
kural sil
ufw delete allow PORT
Belirtilen port için oluşturulan kuralı tamamen sil.
UFW'yi devre dışı bırak
ufw disable
dikkat
Firewall'u tamamen kapat. Sorun giderme dışında kullanma.
