Statik Site Deploy

burakkirlak.info ve tracker.burakkirlak.info gibi projeleri yayınlama

ℹ
Aktif siteler: burakkirlak.info → /var/www/burakkirlak  |  tracker.burakkirlak.info → /var/www/tracker
Yeni Subdomain Ekleme — Adım Adım
1. DNS A kaydı ekle (Hosting Dünyam paneli)
Tür: AAd: subdomainDeğer: 213.142.148.165
DNS değişikliği yayılması 1-30 dakika sürebilir.
2. Repo'yu VPS'e clone et
cd /var/www && git clone https://github.com/HanMiyazaki/repo-adi.git
GitHub reposunu /var/www altına clone et.
3. Nginx config oluştur
nano /etc/nginx/sites-available/site-adi # İçerik: server { listen 80; server_name sub.domain.com; root /var/www/klasor; index index.html; location / { try_files $uri $uri/ =404; } }
Config dosyasını oluştur ve içini doldur.
4. Aktif et ve SSL al
ln -s /etc/nginx/sites-available/site /etc/nginx/sites-enabled/ nginx -t && systemctl reload nginx certbot --nginx -d sub.domain.com
Siteyi aktif et, Nginx'i yenile, SSL sertifikası al.
5. Güncelleme (sonraki değişikliklerde)
cd /var/www/klasor && git pull
Statik sitede servis yenilemeye gerek yok. git pull yeterli.
