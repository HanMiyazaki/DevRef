Linux Temelleri

Günlük kullanımda işe yarayan temel Linux komutları

Dosya & Klasör
içeriği listele
ls -la
Gizli dosyalar dahil tüm dosyaları izinleriyle listele.
klasör gir
cd /var/www/dokushoka
Belirtilen dizine geç.
dosya içeriğini oku
cat dosya.txt
Dosyanın tüm içeriğini terminale yazdır.
dosya düzenle
nano dosya.txt
Nano editörde aç. Kaydet: CTRL+X → Y → Enter.
klasör oluştur
mkdir -p /var/www/yeni-klasor
Klasör oluştur. -p ile ara klasörler de oluşturulur.
dosya kopyala
cp kaynak hedef
Dosyayı kopyala. Klasör için cp -r kullan.
dosya taşı / yeniden adlandır
mv eski-ad yeni-ad
Dosyayı taşı veya adını değiştir.
dosya sil
rm dosya
geri alınamaz
Dosyayı sil. Klasör için rm -rf kullan — dikkatli ol.
Paket & Sistem
paket kur
apt install paket-adi -y
Ubuntu'ya paket kur. -y ile onay sormadan devam eder.
paket listesini güncelle
apt update
Paket listesini yenile. Kurulumdan önce çalıştır.
metin içinde ara
grep -r "aranacak" /var/www/
Belirtilen dizinde tüm dosyalarda metin ara.
process öldür
kill -9 PID
PID numarasını ss -tlnp veya ps aux ile bul.
